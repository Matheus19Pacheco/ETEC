# ETEC
Repositório ETEC
Meu Primeiro Projeto no GitHub

Este é um repositório inicial criado para aprender o fluxo de trabalho com **Git** e **GitHub**. Aqui, coloquei em prática conceitos como criação de repositórios, configuração de `.gitignore`, escolha de licenças e o uso de **Tokens de Acesso Pessoal (PAT)** para segurança.

## 💻 Sobre o Projeto
O projeto consiste em uma página web simples (Landing Page) feita em HTML e CSS puro, servindo como marco inicial dos meus estudos em desenvolvimento.
## 🛠️ Tecnologias Utilizadas
* **HTML5**: Estrutura da página.
* **CSS3**: Estilização básica e centralização com Flexbox.
* **Git**: Controle de versão.
* **Git Bash**: Terminal utilizado para os comandos.

## 📝 O que eu aprendi
Neste processo, aprendi a:
1. Configurar minha identidade no Git (`user.name` e `user.email`).
2. Utilizar o **Git Bash** para navegar entre pastas.
3. Criar commits e enviar arquivos via terminal usando **Tokens**.
4. Diferenciar licenças de software (como a MIT).
